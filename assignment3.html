3. Creating effective visualizations

In this project, three informative visualizations about malaria is created using Python in a Jupyter notebook.

Plotly

Plotly is a Python graphing library that makes interactive, publication-quality graphs.

For this assignment, I choose to use plotly as the major library tool. There're couple of reasons to choose plotly.

  • Primarily, Plotly allows users to interact with graphs on display. Compared to other visualization packages, Plotly allows users to customize interactive visuals by adding features like buttons, sliders, and dropdowns to display different perspectives of graphs.
  • Moreover, Plotly is a library compatible with R, a programming language that I'm familiar with for data visualization. The dash features of Plotly helps to create web applications backed up by Flask, Plotly.js, and React.js in a straightforward way.
  • Finally, a huge range of graphs and charts can be plotted using Plotly, including 3D charts, geospatial charts, and other statistical graphs suitable for the malaria data use case.
