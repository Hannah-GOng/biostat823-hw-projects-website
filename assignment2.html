2. Number theory and a Google recruitment puzzle

Find the first 10-digit prime in the decimal expansion of 17π.

Question Prompt

Find the difference between the sum of the squares of the first n natural numbers and the square of the sum.

The first 5 digits in the decimal expansion of π are 14159. The first 4-digit prime in the decimal expansion of π are 4159. You are asked to find the first 10-digit prime in the decimal expansion of 17π. First solve sub-problems (divide and conquer):

  • Write a function to generate an arbitrary large expansion of a mathematical expression like π. Hint: You can use the standard library decimal or the 3rd party library sympy to do this
  • Write a function to check if a number is prime. Hint: See Sieve of Eratosthenes
  • Write a function to generate sliding windows of a specified width from a long iterable (e.g. a string representation of a number)

    • Solution

    STEP ONE: Expansion Generator: generate an arbitrary large expansion of a mathematical expression

  • 1. Install third party package sympy and mpmath.
  • 2. Generate n number of digits of a mathematical expression using mp.dps = n and sym.Float().

    • STEP TWO: Create a function to check if a number is prime.

  • 1. Set all flag = True.
  • 2. Check if the 'num' is smaller than one, if yes, flag False since prime number > 1.
  • 3. Loop over numbers started from 2; check if there're dividers of 'num'; If yes, then flag False since the prime number is a natural number greater than 1 that is not a product of two smaller natural numbers.

    • STEP THREE: function to generate sliding windows of a specified width from a long iterable and check for prime numbers.

  • 1. Using for loop, create a moving window function to capture 10-digit numbers in the expansion of 17pi.
  • 2. For every number, use the is_prim fuction created previous to check if the number is prime.
  • 3. Break the loop immediately and return 'current' window once the first prime is found.

    • Unit Test

    Test for is_prime

  • 1. Edge cases where the input is smaller or equal than one.
  • 2. Other common inputs with both prime and non-prime numbers

    • Test for find_prime

  • 1. Edge cases where the target prime is not exist.
  • 2. Given that the first 10-digit prime in the expansion e is 7427466391.
